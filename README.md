# OrderFlow — E-Commerce Order Management System

## About
Backend REST API for managing orders, products and customers 
in an e-commerce platform.

## Tech Stack
Java · Spring Boot · MySQL · Spring Data JPA · Git

## Features
- RESTful APIs for Orders, Products and Customers
- Pagination and filtering on large datasets
- Relational MySQL database schema
- Git version control with feature-branch workflow
